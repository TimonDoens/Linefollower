# draadloze communicatie proof of concept
minimale hard- en software waarmee aangetoond wordt dat duplex kan gecommuniceerd worden tussen de microcontroller en een [laptop|smartphone] (schappen wat niet past), gebruik makend van [programma] (in te vullen)
<br />
# Concept
Met het programma dat in deze map staat kunnen we via de seriele monitor via de HC-05 van en naar de pc sturen. Op die manier kan je op de computer via een programma genaamd simple serial terminal commando's geven. Belangrijk is dat je eerst verbind met de bluetooth module via de laptop zelf.
